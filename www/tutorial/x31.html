2. Audience

The primary audience for libcam is other machine vision researchers and people who actively develop and use image processing algorithms. Currently, libcam is supported on GNU/Linux and OS X. There are no immediate plans to support Windows XP. libcam is written in C, and so can be used in C or C++ applications.

libcam is written in C using the GObject framework for object-oriented programming. All classes in libcam are derived from GObject, and memory management is handled via GObject reference counting. Additionally, libcam objects make heavy use of the GLib signalling framework to deliver information.

It is not necessary to have a deep understanding of GObject in order to understand libcam. However, familiarity with object-oriented programming techniques and the publish/subscribe method (also known as signals/slots or the Observer design pattern) is assumed.
