convert.to_rgb8

Convert to 8-bit RGB — Convert almost anything to RGB

Description

convert-to-rgb8 tries to convert its input to a 24-bit RGB image (8-bits per channel). This unit is actually a wrapper around other conversion units, and invokes them as necessary. The primary purpose of this unit is to allow a user to convert an image to RGB if it can be done so using Camunits, without requiring the user to know exactly which unit has the needed conversion routines.

If the input to this unit is already RGB 24bpp, then it is simply passed through.

Table 2. Supported Inputs

Input Formats Notes
BGR 24bpp
BGRA 32bpp
Gray 8bpp
YUYV

Uses convert.colorspace internally.

JPEG

In order of preference, tries to use one of the following units:

ipp.jpeg_decompress
framewave.jpeg_decompress
convert.jpeg_decompress

ipp.jpeg_decompress requires the non-free Intel IPP library. framewave.jpeg_decompress requires the Framewave library. Both IPP and Framewave plugins are distributed with camunits-extra.

Bayer BGGR
Bayer RGGB
Bayer GBRG
Bayer GRBG

Uses convert.fast_debayer internally.
