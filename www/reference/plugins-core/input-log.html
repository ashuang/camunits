Description

input.log provides units that can read images from a Camunits log file. Various features of the log playback can be controlled, such as the playback speed, frame skipping, and looping.

Input Formats

None. This is an input unit.

Output Formats

The output format is determined by the log file

Mode

There are two auto-advance modes, and behave differently only if the image processing chain handles images more slowly than they are produced by the log input unit. In the first mode (0), the log input unit will delay playback of the next frame, and never skips frames. In the second mode (1), the log input unit will skip frames to maintain a "realtime" playback effect.
