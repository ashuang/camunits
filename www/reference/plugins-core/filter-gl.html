filter.gl

OpenGL renderer — Renders images as OpenGL textures

Description

filter.gl renders images as OpenGL textures. It does not modify images.

Input Formats

The following input formats can be rendered by this unit:

BGR 24bpp
RGB 24bpp
RGBA 32bpp
BGRA 32bpp
Bayer RGGB
Bayer BGGR
Bayer GBRG
Bayer GRBG
Gray 8bpp
Big-Endian Gray 16bpp
Little-Endian Gray 16bpp
Native-Endian Float 32bpp

Output Formats

The input format is passed through.

Controls

This plugin does not have any controls.
