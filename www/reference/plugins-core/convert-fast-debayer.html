Description

convert.fast_debayer is a Bayer demosaic filter that is optimized with SSE2 and SSE3 instructions, so it should be pretty fast.

The method used is described by H.S. Malvar et al[1].

Do _NOT_ try to use this filter if your architecture doesn't support SSE2/SSE3.

Input Formats

Bayer RGGB
Bayer BGGR
Bayer GBRG
Bayer GRBG
Gray 8bpp

Output Formats

BGRA 32pp
Gray 8bpp

Controls

Tiling

Sets the Bayer tiling pattern used by the demosaic filter. This is automatically set if the input format is a Bayer format. If the input format is Gray 8pp, then this control must be manually set.
