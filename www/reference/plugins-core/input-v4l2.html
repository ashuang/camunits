input.v4l2 provides access to Video 4 Linux 2 devices.

Requirements

  • GNU/Linux

Input Formats

None. This is an input unit.

Output Formats

The exact list of output formats supported by a unit varies with each camera.

Controls

The input.v4l2 plugin exposes most of the controls for a V4L2 device. The exact number, name, and type of each control depends on the device, as different V4L2 devices (and drivers) expose different controls.

The full list and description of V4L2 controls can be found in the V4L2 API reference. As of March, 2009, this document is located at http://v4l2spec.bytesex.org/spec/.
