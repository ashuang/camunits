Description

input.dc1394 is a plugin that uses libdc1394 to provide access to IIDC-compliant digital cameras.

Requirements

  • libdc1394 >= 2.0

Input Formats

None. This is an input unit.

Output Formats

The exact list of output formats supported by a unit varies with each camera. In general, the pixel formats will be among the following:

  • Gray 8bpp
  • Bayer RGGB
  • Bayer GBRG
  • Bayer GRBG
  • Bayer BGGR
  • IYU1
  • UYVY
  • IYU2
  • RGB 24bpp
  • Big-Endian Gray 16bpp
  • Big-Endian RGB 48bpp
  • Big-Endian signed Gray 16bpp
  • Big-Endian signed RGB 48bpp

Controls

The input.dc1394 plugin exposes most of the control registers for a DC1394 camera. The exact number, name, and type of each control depends on the camera, as different DC1394 cameras expose different controls.

The full list and description of DC1394 control registers can be found in the IIDC specifications. A copy of those specifications can be found at: http://damien.douxchamps.net/ieee1394/libdc1394/iidc/
