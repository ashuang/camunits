Description

output.logger writes images and metadata to a Camunits log file.

Input Formats

All input formats are accepted.

Output Formats

The input format is passed through.

Controls

Filename

Controls the desired filename of the output file.

id:

 desired-filename

type:

 string

Enable Filename Auto Suffix

If this is enabled, then the logger unit will never overwrite existing files. Instead, it will automatically append a suffix to the desired filename such that a new file is always created. If this is not enabled, then Camunits will overwrite existing logfiles, and will use the string specified in the filename control directly for the output file.

id:

 auto-suffix-enable

type:

 boolean

Record

Controls whether or not the logger unit is actually logging frames to disk. The logger unit starts writing to the beginning of a file each time this control is enabled. Specifically, if auto-suffix-enable is set, then each time this control is set to True, then a new log file is created. If auto-suffix-enable is not set, then each time this control is enabled, the log file is truncated and started over.
