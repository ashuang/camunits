libcam Reference Manual

Units
CamUnit — CamUnit is the fundamental object in libcam.
CamUnitFormat — Describes how the data of a CamUnit's input and output buffers are formatted.
CamUnitControl — User adjustable / automatically changing control value.
CamUnitDriver — Factory-like class that instantiates CamUnit objects.
CamFrameBuffer — Container class for a raw data buffer.
Image Processing Chains
CamUnitChain — A sequence of CamUnit objects that form an image processing chain.
CamUnitManager — Factory-like class that manages a collection of CamUnitDriver objects.
