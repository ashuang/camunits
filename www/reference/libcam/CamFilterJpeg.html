Description

CamFilterJpeg can compress RGB, BGRA, and grayscale images to JPEG. It can also decompress JPEG-compressed images to RGB. CamFilterJpeg uses libjpeg.

CamFilterJpeg is a core unit, and is always available with unit id "convert.jpeg"
