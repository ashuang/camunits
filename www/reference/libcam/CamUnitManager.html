Description

CamUnitManager is a factory-like class that provide a (hopefully) convenient way to list and instantiate all available units. A single CamUnitDriver is responsible only for describing and instantiating units of a specific type, whereas a CamUnitManager attempts to aggregate all of the information provided by all known CamUnitDriver objects into a single object.
