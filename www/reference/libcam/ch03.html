libcam Reference Manual

Built-in Units

CamV4L2 — input unit for Video 4 Linux 2 (V4L2) devices
CamV4L — input unit for Video 4 Linux (V4L) devices
CamInputExample — Debugging class, and also demonstration of how to create a simple CamUnitDriver and CamUnit
CamInputLog — input unit for libcam log files
CamConvertToRgb8 — Convenience unit that attempts to convert everything to 24-bit RGB
CamFilterGL — passthrough filter that can render to an OpenGL context
CamFilterJpeg — unit to encode and decode JPEG images
CamFastBayerFilter — SSE2/SSE3 optimized Bayer demosaic filter.
CamColorConversionFilter — General-purpose colorspace conversion
CamLoggerUnit — logging unit that streams its input to disk
