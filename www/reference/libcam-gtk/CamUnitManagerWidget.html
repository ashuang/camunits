Signal Details

The "unit-description-activated" signal

void                user_function                      (CamUnitManagerWidget *manager_widget,
                                                        CamUnitDescription   *udesc,
                                                        gpointer              user_data)           : Run First

The unit-description-activated signal is emitted when the user double-clicks on a unit description

manager_widget : CamUnitManagerWidget emitting the signal
udesc : the CamUnitDescription that was activated
user_data : user data set when the signal handler was connected.
