CamFilterGL — passthrough filter that can render to an OpenGL context

Description

CamFilterGL is a filter subclass of CamUnit that passes images through unchanged, but implements the draw_gl_init, draw_gl, and draw_gl_shutdown routines that allow it to render images to an OpenGL context.

CamFilterGL is a core unit, and is always available with unit id "output.opengl"
